# Project Documentation

This documentation describes the football offside detection project.

## Overview

The offside detection project aims to automate the determination of offside positions in football matches using computer vision techniques.

## Features
- Real-time offside detection
- Integration with video feeds
- User-friendly interface for referees

## Installation
To install the project, clone the repository and run the following command:

```bash
pip install -r requirements.txt
```

## Usage
To run the application, use:

```bash
python main.py
```

# Football Offside Detection Project

This project focuses on detecting offside in football videos using computer vision techniques, deep learning models, and video processing methodologies.

## Project Structure

- **data/**: Contains raw, processed data, and annotations.
  - **raw/**: Raw video data
  - **processed/**: Frames processed from the raw video
  - **annotations/**: Label data for analysis
- **models/**: Contains model definition files.
  - **player_detection/**: Models related to player detection.
  - **ball_detection/**: Models for detecting the ball.
  - **offside_detection/**: Models specifically for offside detection.
- **src/**: Source code for the project.
- **app/**: Web application code (using Streamlit).
- **tests/**: Unit and integration tests for the project.
- **docs/**: Documentation related to the project activities.

## Requirements
- Python 3.x
- Dependencies listed in `requirements.txt`

## Getting Started
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Run the application
